# Ruboty::Homete

A ruboty handler to praise members.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-homete'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-homete

## Usage

```
@ruboty homete - Praise you.
```
